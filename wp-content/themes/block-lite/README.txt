== Description ==

The Block Lite theme features a modern and responsive design with a block style layout for blog posts. It's developed to work seamlessly with the Gutenberg editor and custom blocks. The theme options are located within the WordPress customizer for a simple setup process.

== Installation ==

1. Upload the theme by navigating to Appearance > Themes > Install Themes within the WordPress admin. Select the theme zip file to upload.
2. Activate the theme after uploading.
3. Configure and save the theme options within Appearance > Customize.
